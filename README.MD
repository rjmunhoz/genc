**Instalação**

`npm i -g genc`

**Uso**
- Gerar CPF: `genc f`
- Gerar CNPJ: `genc j`

**Parâmetros**
- Tipo de documento (Obrigatório): `f | j` 
- Quantidade (Opcional): Qualquer valor acima de 1
- Lista (Opcional): `(y | s) | n`;
    - Caso true, não exibirá nenhum texto além dos documentos gerados;
